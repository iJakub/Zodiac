## Description

Simple script to quickly find a person's website using
* Name **(required)**
* Surname **(required)**
* Second Name
* Second Surname
* Nickname

Using this data, the script is able to create 150 urls that may belong to the person you are interested in.

## Executing

Requires ```python``` **(version 3.x)** to run
```
python zodiac.py
```
